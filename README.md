# Todos-react

## 0x00 介绍

使用create-react-app构建，基于react-redux，react-router实现简单的页面跳转，路由，状态管理和待办事项功能。

## 0x01 启动

依次执行以下命令：

```command
$ cd todos-react
$ npm install
$ yarn start
```

## 0x02 路由

react-router@4 <https://reacttraining.com/react-router>

## 0x03 状态管理

react-redux <https://github.com/reduxjs/react-redux>
